Welcome to GameGrid Plugin APIs











     Hello, we see you have interest in learning about the GG Plugin API, or what it takes to create a GameGrid Plugin. Plugins are unofficial GameGrid sectors, built by other developers, that can easily be installed to GameGrid. Before you start scouring the Plugin API, read the API Welcome Page, and follow their instructions to get started. When done with that, read the below paragraphs.

THE PLUGIN API


      You should know about eval, rescue, variables, etc in Ruby. These are basic concepts that can be used for complex things. Now, Plugins are completely built in Ruby. You should've already toured the GameGrid.rb file. See all the variables, like $lives, $energy, $dead, $battle, etc? Also, see all the methods, like checkdeads(), attack(), evoke() and more? You can use all these and more, in your Ruby plugin program. To start out, create a new file called myplugin.rb in a good code editor. Now write this code: puts 'This is the awesomest plugin ever! Yeah! Oh, you have #{$lives} lives!', which just prints out a random sentence, which also shows how many lives you have in the GameGrid game! Now, copy everything. Go to the plgvaldb.rb file. This is the database of all valid plugin data. You should see a hash "{". Add a parameter to the hash, with your plugin name, myplugin, and the ruby script. It should look like this:
{ "install"=>"puts 'The GG Install plugin is located in the install.rb file. This is the starter plugin for GameGrid.';", "starterplugin"=>"puts 'Hello world -- Starter plugin'", "myplugin"=>"puts 'This is the awesomest plugin ever! Yeah! Oh, you have #{$lives} lives!'" }
Make sure you remember the quotes around your Ruby script! Okay, now save plgvaldb.rb. Go to the command line, cd to the directory where GameGrid.rb and all it's files are in. Type a random letter and hit enter, three times, to finish the survey quickly. Now type Plugin.install("myplugin") and hit enter to load your plugin. If you got an error message, delete all that you did, and start all over, from the top. If you did not get an error, good! Type Plugin.run() and hit enter to run your plugin. You should see this: This is the awesomest plugin ever! Yeah! Oh, you have 5 lives!
=> nil
If you saw that, great! You've just created your first plugin.

NOTE: Before beginnning, locate the GameGrid.rb file. Also, make sure to open a command line (Command Prompt or Mac Terminal) window and change directories to the folder the GameGrid.rb file is located in. Also, install or open a new window in Google Chrome (www.chrome.com). Chrome is the best tool to test it out. When done, go to the next paragraph.
